# Sales-Trend-Time-Based-Performance

## Project Overview

This project analyzes transactional sales data from Afficionado Coffee Roasters to identify time-based sales patterns, customer purchasing behavior, and store performance using Python data analytics techniques.

## Objectives

- Analyze overall sales performance across 2025.
- Identify hourly sales trends.
- Compare sales performance across store locations.
- Detect peak and off-peak business hours.
- Forecast future sales using ARIMA.
- Develop an interactive Streamlit dashboard.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Scikit-learn
- Statsmodels
- Streamlit

## Features

- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Hourly Revenue Analysis
- Store-wise Comparison
- Heatmap Visualization
- Linear Regression
- ARIMA Forecasting
- Interactive Streamlit Dashboard
